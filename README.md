dotfiles
=========

zsh/vim Setup
-----

  1. Install zsh

  ```
   apt-get install zsh
  ```

  2. Clone this repository

  ```
    git clone https://github.com/ExpLog/dotfiles.git
  ```

  3. Run the setup script
  ```
    ./install_oh_my_zsh.sh
  ```
  4. Install the font

  5. Make `zsh` your default shell

  ```
    chsh -s /bin/zsh
  ```
