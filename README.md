dotfiles
=========

Setup
-----

  1. Install zsh

  ```
   apt-get install zsh
  ```

  2. Clone this repository

  ```
    git clone https://github.com/felipeparaujo/dotfiles.git
  ```
  
  3. Run the setup script
  ```
    ./setup.sh
  ```
  4. Install the font


  6. Make `zsh` your default shell

  ```
    chsh -s /bin/zsh
  ```
